# Integração e análise de dados bibliométricos dos artigos científicos do IF-USP
Trabalho de Formatura do Curso de Bacharelado em Matemática Aplicada e Computacional do IME-USP

## Resumo

A análise de dados bibliográficos de produções intelectuais constitui um importante
instrumento de avaliação acadêmica. A plataforma Lattes, principal fonte de informações de
produções acadêmicas no Brasil, impõe restrições à coleta de dados e não dispõe de dados
de citações nas importações individuais dos dados currículos.
Com o desenvolvimento da ciência de dados, a coleta, o processamento, a integração

e a análise desses dados podem ser automatizadas. Neste trabalho, aplicações foram desen-
volvidos para a coleta automática e integração dos dados da plataforma Lattes e outras bases

de dados bibliométricos: Microsoft Academic, Google Scholar, Scopus e Web of Science.
A técnica Record Linkage foi empregada para a integração dos dados e obtenção
de um único conjunto enriquecido. Foram coletados dados dos docentes credenciados no
programa de pós-graduação do IFUSP. A técnica foi avaliada para diferentes critérios de
comparação e parâmetros e, a configuração ótima obteve a acurácia 0.97 na integração dos
dados pelo título e ano de publicação dos artigos.
Com o conjunto de dados enriquecidos, foi realizada análise exploratória. Foram
analisadas as distorções causadas pela participação de grandes colaborações científicas em
métricas como Índice h. Com os dados enriquecidos, foram calculadas variantes do Índice h
que propõem normalizar os efeitos de artigos múltiplos com autores no cálculo do Índice h.
O melhor resultado foi obtido pelo Índice hf rac.
Palavras-chave: Dados bibliográficos, Record Linkage, índice h.
